Чтобы запустить данный файл необходимо:
1. Скачать программу QT Designer https://build-system.fman.io/qt-designer-download
2. Установить библиотеку pyQT5
3. В папке есть все файлы, необходимые для запуска:
newsform.ui - интерфейс созданной программы в QT Designer
newsform.py - конвертированный интерфейс
news.py - код программы

Я отдельно открываю PyCharm и через file... Open... открываю файл news.py

Все должно работать :) (дай Бог)
